libbench
========

The in-code profiler library.

Directory structure
-------------------

* lib/  
  The core library.

* test/  
  A simple multi-threaded test that generates a profiling file.

* tool/  
  The tools that display the content created by the library. At the moment it includes a GTK 
viewer.
